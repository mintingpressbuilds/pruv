# Installation

pruv is distributed as two Python packages. Install whichever fits your needs.

## Requirements

- Python 3.10 or higher
- No system dependencies required for xycore
- `pip`, `uv`, or `poetry` for package management

## Install xycore (Zero Dependencies)

If you only need the core XY primitive -- chains, entries, verification, and local storage -- install `xycore`. It has **zero external dependencies**.

<CodeGroup>
```bash pip
pip install xycore
```

```bash uv
uv add xycore
```

```bash poetry
poetry add xycore
```
</CodeGroup>

Verify the installation:

```python
import xycore
print(xycore.__version__)  # 1.0.0
```

### Optional: Digital Signatures

xycore supports Ed25519 digital signatures via an optional dependency:

<CodeGroup>
```bash pip
pip install xycore[signatures]
```

```bash uv
uv add "xycore[signatures]"
```

```bash poetry
poetry add "xycore[signatures]"
```
</CodeGroup>

This installs the `cryptography` package (>=41.0) for Ed25519 key generation and verification.

## Install pruv (Full SDK)

For the complete toolkit -- scanning, wrapping, checkpoints, approval gates, CLI, and cloud sync -- install `pruv`:

<CodeGroup>
```bash pip
pip install pruv
```

```bash uv
uv add pruv
```

```bash poetry
poetry add pruv
```
</CodeGroup>

The `pruv` package automatically installs `xycore` plus these additional dependencies:

| Dependency | Purpose |
|-----------|---------|
| `xycore>=1.0.0` | Core XY primitive |
| `httpx>=0.25.0` | Cloud sync and approval webhooks |
| `click>=8.0` | CLI interface |
| `pydantic>=2.0` | Data validation |
| `zstandard>=0.21.0` | Checkpoint compression |

Verify the installation:

```python
import pruv
print(pruv.__version__)  # 1.0.0
```

## Verify CLI Installation

After installing `pruv`, the CLI should be available:

```bash
pruv --version
# pruv, version 1.0.0
```

Test with a quick scan:

```bash
pruv scan .
# pruv scan: .
#   Files: 42
#   Lines: 3847
#   Hash:  a1b2c3d4e5f6...
#   Frameworks: fastapi, react
#   Services: postgresql, redis
```

## Cloud Setup (Optional)

To sync chains to [pruv cloud](https://app.pruv.dev), set your API key:

```bash
export PRUV_API_KEY=pv_live_your_api_key_here
```

Get your API key from the [pruv dashboard](https://app.pruv.dev). Keys use the prefix `pv_live_` for production and `pv_test_` for development.

<Note>
  You can use pruv entirely offline. Cloud sync is optional and chains work locally without an API key.
</Note>

## Environment Variables

Configure pruv behavior with environment variables:

```bash
# API key for cloud sync
export PRUV_API_KEY=pv_live_your_api_key_here

# Auto-redact secrets from chain states (default: true)
export PRUV_AUTO_REDACT=true

# Auto-checkpoint at intervals (default: false)
export PRUV_AUTO_CHECKPOINT=false
```

See the full list of [environment variables](/reference/environment-variables).

## Development Installation

To work on pruv itself, clone the repo and install in development mode:

```bash
git clone https://github.com/mintingpressbuilds/pruv
cd pruv

# Install xycore in development mode
pip install -e packages/xycore

# Install pruv in development mode
pip install -e packages/pruv

# Run tests
pytest packages/xycore/tests
pytest packages/pruv/tests
```
